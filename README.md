# IronLady-AI_task

💃 Iron Lady AI Career Catalyst & Lead Manager
This project was developed for the Iron Lady AI Internship assignment. It leverages advanced Generative AI to empower women leaders and streamline internal operations.

🚀 Project Overview
This repository contains two primary solutions:

Task 1: Career Catalyst (Customer Facing) – A strategic AI advisor that maps a user's career stage to Iron Lady's specific leadership programs using the "Business War Tactics" philosophy.

Task 2: Internal Lead Manager (Admin) – A lead tracking system with AI-powered candidate scoring to help the operations team prioritize high-potential nominees.

🛠️ Technical Stack
Framework: Streamlit.

AI Engine: Google Gemini 2.5/3 Flash (via Google AI Studio).

Orchestration: LangChain (Google GenAI).

Data Handling: Pandas & CSV.

✨ Features
Task 1: AI Career Catalyst
Context-Aware: Trained on Iron Lady’s "Unapologetic Winning" and "Business War Tactics" values.

Program Mapping: Recommends specific programs like Leadership Essentials or Master of Business Warfare based on years of experience and career goals.

Task 2: Internal Lead Manager
CRUD Functionality: Allows the team to Create, Read, and Update lead information in real-time.

AI Scoring: Automatically generates a "Potential Score" for every new lead based on their professional profile.

Installation & Setup
Clone the repository:
git clone https://github.com/goku2710/IronLady-AI_task.git

Install dependencies:
pip install -r requirements.txt

Run the Apps:

For Task 1: streamlit run app.py.

For Task 2: streamlit run internal_app.py.
